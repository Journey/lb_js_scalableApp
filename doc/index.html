Documentation of the
Scalable JavaScript Application framework
by Legalbox

The Scalable JavaScript Application framework is an open-source JavaScript library intended for the development of single-page Web applications.
It is based on the Scalable JavaScript Application Architecture by Nicholas C. Zakas.

This is the entry point for the documentation of the library.
You should start by:

  1. Watching the presentation by Nicholas Zakas: read the slides / watch the video (transcript included).

  2. Reading the design principles exposed in Design Patterns for Scalable JavaScript Application by Legalbox.
    This is a document in Open Document Format, which can be read e.g. with OpenOffice or LibreOffice.

  3. Read the API documentation of the library.
    This HTML documentation is generated from comments in the source code.
    You need to run the build.xml with Ant to generate it.
    This requires Apache Ant, a Java JDK/OpenJDK and perl/ActivePerl.

    The public API is made of:

    • the Sandbox API, for use in application modules
    • the application core API, for use in your application main script, to load and start modules
